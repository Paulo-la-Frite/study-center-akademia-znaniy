# Информационная система учебного центра «Академия Знаний»

Стек: **Python + Tkinter + PostgreSQL**

## О проекте

ИС автоматизирует ключевые процессы учебного центра дополнительного профессионального образования:
- Ведение реестра слушателей
- Учёт успеваемости и посещаемости
- Формирование расписания
- Управление пользователями (авторизация, роли, блокировки)

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| Администратор | Управление пользователями, снятие блокировок |
| Методист | Формирование расписания, отчёты |
| Преподаватель | Выставление оценок, отметка посещаемости |
| Студент | Просмотр оценок и расписания |

## Безопасность

- Пароли хэшируются bcrypt (12 раундов)
- Блокировка после 3 неверных попыток
- Автоблокировка при неактивности > 30 дней
- Принудительная смена пароля при первом входе
- Защита от SQL-инъекций (параметризованные запросы)

## Установка и запуск

1. Установить PostgreSQL 15+
2. Создать базу данных `study_center`
3. Выполнить скрипты из `database/`
4. Установить зависимости: `pip install psycopg2-binary bcrypt`
5. Настроить `src/config.py` (указать пароль от БД)
6. Запустить: `python -m src.main`
